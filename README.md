# Enterprise-Data-Warehouse-Databricks-dbt-Power-BI
Este repositório documenta a arquitetura de dados e a modelagem analítica do Banvic, um projeto prático focado em resolver problemas de negócios no setor bancário. O objetivo principal é demonstrar o ciclo de vida completo dos dados, desde a ingestão bruta até a disponibilização para consumo em painéis de Business Intelligence (BI).
